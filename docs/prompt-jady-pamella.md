# Jady Pamella Portfolio - Test Prompt

Use this prompt to test the Page Selector feature with a multi-page portfolio.

---

## Prompt

```
Create a multi-page professional portfolio website for Jady Pamella, a Senior Technology Manager & AI Expert, with the following pages:

1. index.html - Home/Landing page with:
   - Navigation bar with links to all pages (About, Projects, Experience, Contact)
   - Hero section with ID "hero":
     * Name "Jady Pamella" as main headline
     * Title "Senior Technology Manager | AI & Digital Transformation Expert"
     * Brief intro text
     * Profile photo placeholder (circular)
     * Social links (LinkedIn, GitHub, Email icons)
   - About section with ID "about":
     * Professional summary
     * Key skills in tags/badges
   - Stats section with ID "stats":
     * Years of experience: 10+
     * Projects delivered: 50+
     * Team members led: 20+

2. projects.html - Projects page with:
   - Section "Featured Projects" with ID "featured":
     * Grid of 4 project cards
     * Each card: image placeholder, title, description, tech stack tags, link
   - Section "Open Source" with ID "opensource":
     * 3 cards for open source contributions

3. experience.html - Experience page with:
   - Section "Work Experience" with ID "work":
     * Timeline layout with 4 positions
     * Each: company, role, dates, bullet points
   - Section "Education" with ID "education":
     * 2 education entries
   - Section "Certifications" with ID "certifications":
     * Grid of certification badges/cards

4. contact.html - Contact page with:
   - Section "Get in Touch" with ID "contact-form":
     * Contact form (name, email, subject, message)
     * Submit button
   - Section "Connect" with ID "social":
     * Social media links with icons
     * Email and location info

Design Requirements:
- Profile image: https://jadypamella.com/images/profile.png
- Color palette:
  * Rosa destaque (highlight pink): #DF3D5C - Used for top menu, titles, icons and highlight elements
  * Preto grafite (graphite black): #232222 - Used for left side menu and dark backgrounds
  * Branco quase puro (almost pure white): #FBF9F9 - Used for main card backgrounds and content areas
- Modern glassmorphism cards with blur and subtle borders
- Font: Inter or similar clean sans-serif
- Subtle glow effects on buttons and accent elements
- Smooth hover animations
- Fully responsive design
- Do not use emojis anywhere, use icons instead
- The year is 2025
- Responsive design
- Footer on all pages with copyright 2025
- Active nav link highlighting
```

---

## Expected Result

After generation, the Page Selector should show:

| Page | Sections |
|------|----------|
| index.html | hero, about, stats |
| projects.html | featured, opensource |
| experience.html | work, education, certifications |
| contact.html | contact-form, social |

---

## Design Reference

The design follows jadypamella.com with:
- Graphite black (#232222) for dark backgrounds and side menu
- Pink (#DF3D5C) accents for highlights, titles, and icons
- Almost pure white (#FBF9F9) for card backgrounds and content areas
- Glassmorphism card effects
- Clean typography
- Professional tech portfolio aesthetic
